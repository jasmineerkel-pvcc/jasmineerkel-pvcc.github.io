Operating Systems Terminology

TERM DEFINITION
Operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
Job Queue In batch processing systems, the jobs residing in mass storage wait for execution in a job queue.
FIFO Abbreviation for first-in, first-out. It is the fashion that queues are ordered.
Interactive Processing It allows a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing A feature that provides service to multiple users at the same time.
Multiprogramming A tequnique used to implement time-sharing, in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
Application Software Smartphone application software is rapidly a expanding collection.
System Software The advent of computer networks in which numerous machines are connected over great distances has led to the creation of software systems to coordinate the network’s activities.
Utility Software Within the class of system software are two categories: one is the operating system itself and the other consists of software units collectively known as utility software. The majority of an installation’s utility software consists of programs for performing activities that are fundamental to computer installations but not included in the operating system.
Graphical User Interface (GUI) Objects to be manipulated, such as files and programs, are represented pictorially on the display as icons.
Shell Older user interfaces, called shells, communicated with users through textual messages using a keyboard and monitor screen.
Kernel The internal part of an operating system. An operating system’s kernel contains those software components that perform the very basic functions required by the computer installation.
Device driver The software units that communicate with the controllers (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine.
File Manager Coordinates the use of the machine’s mass storage facilities.
Memory Manager Charged with the task of coordinating the machine’s use of main memory.
Virtual Memory The large "fictional" memory space created by paging.
Boot Strapping A procedure that transfers the operating system from mass storage (where it is permanently stored) into main memory (which is essentially empty when the machine is first turned on).
ROM Read-only memory, or ROM, is a memory whose contents can be read but not altered.
Deadlock A problem that can arise during resource allocation. It is a condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling The technique of holding data for output at a later but more convenient time is called spooling.
Super User (or Administrator) Can remove (kill) some of the processes if a deadlock should occur due to a full table.
Multi-Factor Authentication Requirement that more than one piece of evidence is needed to gain access.
